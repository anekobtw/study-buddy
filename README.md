## Getting Started

### Environment
```bash
cp ../.env.example ../.env   # PLEASE CHANGE THE VARIABLES' VALUES
```

### Backend Setup

```bash
cd backend
poetry install
poetry run python main.py
```

The backend will run on `http://localhost:8000`

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

The frontend will run on `http://localhost:5173`
